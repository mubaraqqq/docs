--header 'Content-Type: application/json' \
--header 'Authorization: Bearer ACCESS_TOKEN' \
--data-raw '{
  "name": "Transaction Drive",
  "coinValue": "10",

### URLs

```bash Get All (Uses default limit and page)
https://test-api.bvndle.com/api/v1/laas/rules
```

```bash Get All with your limit and Page
https://test-api.bvndle.com/api/v1/laas/rules?limit=100&page=1
```

```bash Get Active Rules
https://test-api.bvndle.com/api/v1/laas/rules?limit=100&page=1
```

```bash Get Inactive Rules
https://test-api.bvndle.com/api/v1/laas/rules?limit=100&page=1
```
