###

```bash cURL
curl --request PUT
'https://test-api.bvndle.com/api/v1/laas/streaks/998bd723-3d40-48d4-a896-b7eca283aafe/rewards' \
--header 'Content-Type: application/json' \

{
    "rewards": [
        {
            "ticketId": "36a2ed79-42fd-4ea7-b558-a4869adf726f",
            "type": "ticket"
        }
    ]
}

```
