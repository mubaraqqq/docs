###

```bash cURL
curl --request POST
'https://test-api.bvndle.com/api/v1/laas/streaks' \
--header 'Content-Type: application/json' \

{
    "name": "daily_transaction_streak",
    "rewards": [
        {
            "ticketId": "36a2ed79-42fd-4ea7-b558-a4869adf726f",
            "type": "ticket"
        },
        {
            "amount": 2000,
            "type": "airtime"
        }
    ]
}

```
