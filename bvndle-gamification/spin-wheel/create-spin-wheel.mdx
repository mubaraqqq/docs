###

```bash cURL
curl --request POST
'https://test-api.bvndle.com/api/v1/laas/spin-wheel' \
--header 'Content-Type: application/json' \

{
    "transactionPin": "1234",
    "maxSpins": -1,
    "name": "Another spin wheel",
    "entryFees": [
        {
            "fee": 5,
            "type": "coins"
        },
        {
            "fee": 200,
            "type": "airtime"
        }
    ]
}

```
