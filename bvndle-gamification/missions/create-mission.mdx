###

```bash cURL
curl --request POST
'https://test-api.bvndle.com/api/v1/laas/missions' \
--header 'Content-Type: application/json' \

{
    "name": "20K Yearly data purchase",
    "description": "Recharge data daily",
    "pointsReward": 1,
    "startDate": "2025-08-06",
    "endDate": "2025-08-15",
    "transactionPin": "1234",
    "requirements": [
        "Finish your airtime",
        "Recharge a new batch of airtime"
    ]
}

```
